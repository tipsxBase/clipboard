使用指南

基础操作​

唤出界面​

默认情况下，您可以使用以下全局快捷键唤出剪贴板历史面板：

  • macOS: Cmd + Shift + V
  • Windows / Linux: Ctrl + Shift + V

提示: 您可以在应用的设置页面修改此快捷键。

列表导航​

为了提高效率，我们支持多种导航方式，让您的双手无需离开键盘：

  • 上下移动:
    • 方向键 ↑ / ↓
    • Vim 风格 j (下) / k (上)
    • Emacs 风格 Ctrl+n (下) / Ctrl+p (上)
  • 确认/粘贴: 按下 Enter 键将选中的内容粘贴到当前活动窗口，并自动关闭面板。
  • 预览详情: 按下 Space (空格键) 查看当前选中项的详细信息（适用于长文本或大图）。再次按下 Space 关闭预览。
  • 关闭面板: 按 Esc。

高级功能​

粘贴队列 (Paste Stack)​

粘贴队列允许你一次性复制多个项目，并按顺序依次粘贴。这在填写表单或在应用程序之间移动数据时非常有用。

  1. 选择项目:
    • 键盘: 导航到项目并按 x 键进行选择/取消选择。
    • 鼠标: 按住 Cmd (macOS) 或 Ctrl (Windows) 并点击项目。
    • 选中的项目上会出现数字标记（如 ①, ②），指示粘贴顺序。
  2. 启动队列: 按 Enter 键。第一个项目将被复制到剪贴板，窗口关闭。
  3. 按顺序粘贴:
    • 粘贴第一个项目 (Cmd+V)。
    • 按下 全局快捷键 (默认 Cmd+Shift+V)，程序会自动将队列中的 下一个 项目复制到剪贴板。
    • 重复此操作直到队列为空。

OCR (文字识别)​

注意: 目前支持 macOS 和 Windows。

你可以直接从剪贴板历史记录中的图片提取文字。

  1. 在列表中选中一个图片项目。
  2. 按 Space 键打开预览。
  3. 点击右下角的 "提取文字" 按钮。
  4. 提取的文字将自动复制到剪贴板，并作为一个新的文本项目添加到历史记录中。

紧凑模式​

如果你喜欢更紧凑的信息视图，可以在设置中开启 紧凑模式。该模式会：

  • 减小每个列表项的高度。
  • 在列表中隐藏图片预览（显示 [Image] 文本）。
  • 在屏幕上显示更多项目。

功能特性​

置顶项目​

您可以将重要的项目置顶显示。置顶的项目将：

  • 始终显示在历史记录列表的顶部。
  • 不会被自动删除，即使达到了历史记录数量限制。

要置顶一个项目，请将鼠标悬停在项目上并点击置顶图标。

敏感数据​

您可以将项目标记为“敏感”。敏感项目将：

  • 在数据库中加密存储。
  • 在界面上模糊显示，直到鼠标悬停。

注意: 为了防止误报，基于熵值的自动敏感内容检测已被移除。您仍然可以在设置中配置“敏感应用”，以忽略来自特定应用（如密码管理器）的剪贴板内容。

搜索功能​

面板顶部设有搜索框。打开面板后，直接输入关键词即可过滤历史记录。

  • 支持模糊搜索。
  • 搜索范围包括文本内容和来源应用名称。

数据管理​

存储位置​

为了保障您的数据隐私，所有数据均存储在您的本地设备上：

  • macOS: ~/.clipboard-manager/
  • Windows: C:\Users\<Username>\.clipboard-manager\
  • Linux: ~/.clipboard-manager/

数据库安全​

  • 数据库文件名为 history.db。
  • 数据库使用 AES-GCM 算法进行加密，密钥存储在本地安全位置。这意味着即使有人获取了您的数据库文件，没有密钥也无法读取其中的内容。

隐私设置​

在设置中，您可以：

  • 暂停记录: 暂时停止监听剪贴板变化。
  • 忽略应用: 添加不需要记录的应用名称（如 1Password, KeyChain 等），保护敏感信息。
  • 清空历史: 一键删除所有本地存储的历史记录。
